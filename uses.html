Uses

Gear, software, and the homelab stack. Not a sponsored list — just what I actually reach for.

Editor & CLI

  • VS Code — daily driver for everything that isn't a one-off script
  • Claude Code — AI pair-programming, surprisingly load-bearing in my workflow
  • Git Bash — Unix-ish shell on Windows that doesn't get in the way
  • GitHub CLI (gh) — PRs, issues, API pokes without leaving the terminal

Homelab

  • Small ARM-based home server running Debian — quiet, low-power, always on
  • Container PaaS with reverse proxy — for everything that runs as a service
  • Mesh VPN — private-side access to the lab when I'm away from home
  • Media streaming — for the things I already own
  • Self-hosted password manager
  • Self-hosted note sync — keeps my notes consistent across devices
  • Workflow automation — glues the smaller services together
  • Subscription tracker — so nothing renews behind my back

Stack for this site

  • Plain HTML/CSS/JS — no build step, no framework, no regrets
  • GitHub Pages — hosting, free, auto-deploys on push to main
  • Cloudflare DNS + Redirect Rules — points the domain, redirects the typo
  • Cloudflare Workers + KV — backs the arcade's global leaderboard
  • marked.js — markdown → HTML for journal entries

Gaming

  • PC — where most games happen these days
  • PS5 — exclusives and couch gaming
  • Steam — library, backlog, wishlist of shame
